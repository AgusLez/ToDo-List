My first project with React.js

Created a simple To-do App which lets you create tasks; you are able to change their state to completed or you can simply delete the task. You can also empty your tasks list.

What I learnt:

  • Create a project from scratch
  • Class components and functional components
  • Events and hooks (useState)
  • Difference between var, let and const
  • Truthy expressions
  • Props- destructuring props, spread operator
  • Render list items, each element with a unique key
  • Use of Font awesome
  • Map function
  • npm
  • Version control: Git
